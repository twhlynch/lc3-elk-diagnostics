# LC-3 Elk Diagnostics

VSCode Diagnostics provider for LC-3 Assembly bridging the Elk LC-3 toolchain.

## Features

- Error and warning diagnostics for LC-3 Assembly files, powered by Elk.

- Automatic diagnostics updates on content changes.

- Automatic Installation and updating of Elk if not already present.
